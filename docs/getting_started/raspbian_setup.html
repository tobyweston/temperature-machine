Raspbian Quick Start Guide

Follow these steps to get up and running with a fresh install of Raspbian Lite, all ready for the temperature-machine.

  1. Download the image of Raspbian Stretch Lite
  2. On Mac, use Etcher to flash a new SD card with the image. Refer to raspberrypi.org for other platforms.

  3. Setup your wifi by inserting your new SD card to your machine and creating a file wpa_supplicant.conf under /boot. Mine looks like this (the scan_ssid=1 is only needed if you’re using a hidden network). On Mac, it’s under /Volumes/boot.

     ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
 update_config=1
    
 network={
   ssid="ssid"
   scan_ssid=1
   psk="plain text password"
 }
  4. Whilst you’re there, add dtoverlay=w1-gpio to the /boot/config.txt file.
  5. Also run the following touch ssh against the boot folder (e.g. touch /Volume/boot/ssh on Mac)
  6. Unmount and stick the card in the Pi. Power up.
  7. On the Pi (ssh pi@raspberrypi.local), run raspi-config to:
    1. Set the hostname to the room name (garage, study etc)
    2. Expand the file system
  8. Reboot. Follow the install steps.

Extras

You may want some optional extras:

  • To disable the activity light, add the following to /boot/config.txt (tested on the Pi Zero only).

      # disable the activity LED (intended for the Pi Zero)
  dtparam=act_led_trigger=none
  dtparam=act_led_activelow=on

  • Share the disk with Mac OSX

      sudo apt-get install netatalk

  • Set the timezone (for UK)

      sudo cp /usr/share/zoneinfo/Europe/London /etc/localtime

  • Fix vi being weird

      echo 'set nocompatible' > ~/.vimrc
